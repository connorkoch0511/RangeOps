Operator console

The RangeOps desktop operator console — schedule test missions and capture live telemetry. Built with Avalonia / .NET 8. Self-contained, so no .NET install is required.

The console is the operator tool for the RangeOps stack, not a standalone app — on launch it connects to the shared PostgreSQL database and the telemetry sim, so run those first (see the repo README: docker compose up -d db, then start sensor-sim). Builds come from every GitHub release.
